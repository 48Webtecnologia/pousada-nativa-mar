# Pousada Nativamar - Landing Page

Landing page responsiva e otimizada da **Pousada Nativamar** em Guaratuba/PR (bairro Nereidas). Focada em conversão de reservas diretas pelo WhatsApp.

## 🚀 Funcionalidades e Ajustes

* **Otimização de Performance:** Imagens convertidas para o formato moderno `.webp` e compactadas para otimização de velocidade de carregamento (SEO e Mobile-first).
* **Estrutura Profissional de Localização:** Seção de mapas em 2 colunas no desktop com destaques de proximidade (distâncias a pé para praia e comércios).
* **Identidade Visual Limpa:** Logo oficial aplicada com transparência sobre os fundos institucionais da página.
* **CTAs de Conversão:** Botões de reserva direcionando para o WhatsApp oficial da empresa `(41) 98512-1201` com mensagem pré-configurada.
* **SEO Básico:** Título e meta descriptions otimizados para indexação orgânica do Google.

## 🛠️ Tecnologias Utilizadas

* HTML5 (Semântico)
* CSS3 (Design Responsivo e Flexbox/Grid)
* JavaScript Vanilla
